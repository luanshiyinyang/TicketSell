# 简单的django 实现的购票web项目
### 实现功能
   - 用户的注册，登录
   - 非登录用户和登录用户的车票查询
   - 登录用户的购票，退票
   - 登录用户的已购车票(订单）查询
   - API请求支持
### 使用框架
   - django（python）
   - djangorestframework(python)
   - bootstrap（web）
   - requests(python spider)
### 环境需求
   - linux服务器环境,启用80端口重定向
   - python3.x
   - django2.0
   - mysql-server,mysql-client
